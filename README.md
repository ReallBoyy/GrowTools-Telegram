# GrowTools-Telegram
Telegram GrowTools bot.

## Required
- nodejs v22 or higher

## Installation
```bash
npm i
```
or
```bash
npm install
```

## Start Bot
```bash
node app
```

## Features
- Items encoder
- Items decoder
- PNG to RTTEX
- RTTEX to PNG
- Protonhash
- Image resizer
- Plugins/ESM Script based
